## 🧩 Common Error Fixes

Sometimes the Windows installer stops during the process. Try these methods if you encounter problems:

### Fix One: Run as Administrator
Right-click the installation file. Select "Run as administrator." This gives the installer permission to create folders and registry keys that standard access lacks.

### Fix Two: Check Windows Updates
Windows 11 requires current security patches for network software. Go to Settings, select Update & Security, and click "Check for updates." Install any pending items, restart your machine, and launch the installer again.

### Fix Three: Disable Antivirus Temporarily
Anti-malware tools sometimes block software installers. Turn off real-time protection for ten minutes while you install the tool. Remember to turn the protection back on immediately after the installation finishes.

### Fix Four: Check .NET Framework
Ekahau Pro requires specific software libraries to function. If you get an error about missing files, download and install the latest .NET Framework directly from the Microsoft website.
